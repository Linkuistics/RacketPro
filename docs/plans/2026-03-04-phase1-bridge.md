# Phase 1: The Bridge — Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Racket can create a Tauri window, register reactive cells, declare a layout of Lit Web Component primitives, and handle user events — proving the full Racket-driven architecture end-to-end.

**Architecture:** Racket process communicates with Tauri's Rust backend via JSON-RPC over stdin/stdout. Rust routes messages between Racket and the WebView via Tauri IPC. The WebView uses Lit Web Components for primitives and @preact/signals-core for cell→DOM reactivity. No build step — all frontend code is native ES modules.

**Tech Stack:**
- Tauri v2 (Rust backend, WebView frontend)
- Lit 3.x (Web Components, served as local vendor ESM)
- @preact/signals-core 1.x (reactive signals, local vendor ESM)
- Racket 8.x+ (mrracket-core package)
- Vite: NOT used — Tauri serves static files directly

**Step 2: Create bridge.rs — Racket process manager**

Create `src-tauri/src/bridge.rs`:

```rust
use serde_json::Value;
use std::io::{BufRead, BufReader, Write};
use std::process::{Child, Command, Stdio};
use std::sync::{Arc, Mutex};
use tauri::{AppHandle, Emitter, Manager};
use tokio::sync::mpsc;

pub struct RacketBridge {
    child: Mutex<Option<Child>>,
    sender: mpsc::UnboundedSender<String>,
}

impl RacketBridge {
    /// Spawn the Racket process and wire up stdin/stdout.
    /// Messages from Racket are forwarded as Tauri events to the WebView.
    /// Messages to Racket are sent via the `sender` channel.
    pub fn start(app: AppHandle, racket_script: &str) -> Arc<Self> {
        let mut child = Command::new("racket")
            .arg(racket_script)
            .stdin(Stdio::piped())
            .stdout(Stdio::piped())
            .stderr(Stdio::inherit())
            .spawn()
            .expect("Failed to spawn Racket process. Is Racket on PATH?");

        let stdout = child.stdout.take().expect("Failed to capture Racket stdout");
        let stdin = child.stdin.take().expect("Failed to capture Racket stdin");

        let (sender, mut receiver) = mpsc::unbounded_channel::<String>();

        // Reader thread: Racket stdout → Tauri events
        let app_clone = app.clone();
        std::thread::spawn(move || {
            let reader = BufReader::new(stdout);
            for line in reader.lines() {
                match line {
                    Ok(line) if line.trim().is_empty() => continue,
                    Ok(line) => {
                        match serde_json::from_str::<Value>(&line) {
                            Ok(msg) => {
                                let msg_type = msg.get("type")
                                    .and_then(|t| t.as_str())
                                    .unwrap_or("unknown");
                                // Emit to all WebView windows
                                let _ = app_clone.emit(&format!("racket:{msg_type}"), &msg);
                            }
                            Err(e) => {
                                eprintln!("[bridge] Invalid JSON from Racket: {e}: {line}");
                            }
                        }
                    }
                    Err(e) => {
                        eprintln!("[bridge] Racket stdout error: {e}");
                        break;
                    }
                }
            }
            eprintln!("[bridge] Racket process stdout closed");
        });

        // Writer task: channel → Racket stdin
        let bridge = Arc::new(Self {
            child: Mutex::new(Some(child)),
            sender,
        });

        let mut stdin = stdin;
        tokio::spawn(async move {
            while let Some(msg) = receiver.recv().await {
                if let Err(e) = writeln!(stdin, "{}", msg) {
                    eprintln!("[bridge] Failed to write to Racket stdin: {e}");
                    break;
                }
                let _ = stdin.flush();
            }
        });

        bridge
    }

    /// Send a JSON message to the Racket process.
    pub fn send(&self, msg: Value) -> Result<(), String> {
        let json = serde_json::to_string(&msg).map_err(|e| e.to_string())?;
        self.sender.send(json).map_err(|e| e.to_string())
    }

    /// Kill the Racket process.
    pub fn stop(&self) {
        if let Some(mut child) = self.child.lock().unwrap().take() {
            let _ = child.kill();
        }
    }
}

impl Drop for RacketBridge {
    fn drop(&mut self) {
        self.stop();
    }
}
```

**Step 2: Create the JSON-RPC protocol module**

Create `racket/mrracket-core/protocol.rkt`:
```racket
#lang racket/base

(require json racket/port)

(provide send-message!
         read-message
         make-message
         message-type
         message-ref
         start-message-loop)

;; Send a JSON message to stdout (→ Rust bridge)
(define (send-message! msg)
  (write-json msg (current-output-port))
  (newline (current-output-port))
  (flush-output (current-output-port)))

;; Read one JSON message from stdin (← Rust bridge)
(define (read-message [in (current-input-port)])
  (define line (read-line in 'any))
  (cond
    [(eof-object? line) eof]
    [(string=? (string-trim line) "") #f]
    [else
     (with-handlers ([exn:fail? (λ (e)
                                   (eprintf "[protocol] Invalid JSON: ~a\n" (exn-message e))
                                   #f)])
       (string->jsexpr line))]))

;; Construct a message
(define (make-message type . pairs)
  (apply hasheq 'type type pairs))

;; Extract the type field
(define (message-type msg)
  (hash-ref msg 'type #f))

;; Extract a field with optional default
(define (message-ref msg key [default #f])
  (hash-ref msg key default))

;; Main event loop: read messages, dispatch to handler
(define (start-message-loop handler)
  (let loop ()
    (define msg (read-message))
    (cond
      [(eof-object? msg)
       (eprintf "[mrracket] Bridge connection closed\n")]
      [(not msg) (loop)] ; skip empty/invalid
      [else
       (with-handlers ([exn:fail? (λ (e)
                                     (eprintf "[mrracket] Handler error: ~a\n" (exn-message e)))])
         (handler msg))
       (loop)])))
```

**Step 3: Create the reactive cell module**

Create `racket/mrracket-core/cell.rkt`:
```racket
#lang racket/base

(require "protocol.rkt")

(provide define-cell
         cell-ref
         cell-set!
         cell-update!
         all-cells
         register-all-cells!)

;; Internal cell registry
(define cells (make-hasheq))

;; Define a reactive cell. Sends registration to frontend.
(define (make-cell name initial-value)
  (hash-set! cells name initial-value)
  name)

;; Macro for convenient cell definition
(define-syntax-rule (define-cell name initial-value)
  (define name (make-cell 'name initial-value)))

;; Get current value
(define (cell-ref name)
  (hash-ref cells name
    (λ () (error 'cell-ref "Unknown cell: ~a" name))))

;; Set value and notify frontend
(define (cell-set! name value)
  (hash-set! cells name value)
  (send-message! (make-message "cell:update" 'name (symbol->string name) 'value value)))

;; Update with a function
(define (cell-update! name f)
  (cell-set! name (f (cell-ref name))))

;; Get all cells as a hash
(define (all-cells) cells)

;; Send registration messages for all cells to frontend
(define (register-all-cells!)
  (for ([(name value) (in-hash cells)])
    (send-message! (make-message "cell:register"
                                  'name (symbol->string name)
                                  'value value))))
```

**Step 1: Create bridge.js — Tauri IPC wrapper**

Create `frontend/core/bridge.js`:
```javascript
/**
 * Bridge between Tauri IPC and the frontend.
 * Listens for Racket messages (forwarded by Rust) and dispatches events.
 * Sends user events back to Racket via Tauri invoke.
 */

const listeners = new Map(); // type → Set<callback>

/**
 * Register a handler for a Racket message type.
 * @param {string} type - Message type (e.g., "cell:register", "layout:set")
 * @param {function} callback - Handler function receiving the message payload
 */
export function onMessage(type, callback) {
  if (!listeners.has(type)) listeners.set(type, new Set());
  listeners.get(type).add(callback);
}

/**
 * Send an event to Racket via the Rust bridge.
 * @param {string} event - Event name (e.g., "increment", "reset")
 * @param {*} [payload] - Optional event data
 */
export async function dispatch(event, payload) {
  try {
    const { invoke } = await import('@tauri-apps/api/core');
    await invoke('send_to_racket', {
      message: { type: 'event', event, payload: payload ?? null }
    });
  } catch (e) {
    console.error('[bridge] Failed to dispatch:', event, e);
  }
}

/**
 * Initialize the bridge — listen for Tauri events from Rust.
 */
export async function initBridge() {
  try {
    const { listen } = await import('@tauri-apps/api/event');

    // Listen for all racket:* events
    const messageTypes = [
      'cell:register', 'cell:update',
      'layout:set',
      'lifecycle:ready',
      'pong'
    ];

    for (const type of messageTypes) {
      await listen(`racket:${type}`, (event) => {
        const msg = event.payload;
        console.debug(`[bridge] ← ${type}`, msg);
        const handlers = listeners.get(type);
        if (handlers) {
          for (const handler of handlers) {
            try { handler(msg); }
            catch (e) { console.error(`[bridge] Handler error for ${type}:`, e); }
          }
        }
      });
    }

    console.log('[bridge] Initialized, listening for Racket messages');
  } catch (e) {
    console.error('[bridge] Failed to initialize:', e);
  }
}
```

**Step 2: Create cells.js — Signal-based cell registry**

Create `frontend/core/cells.js`:
```javascript
/**
 * Reactive cell registry. Maps Racket cell names to @preact/signals.
 * When Racket updates a cell, the corresponding signal updates,
 * and any Lit components observing it re-render.
 */
import { signal } from '@preact/signals-core';
import { onMessage } from './bridge.js';

/** @type {Map<string, import('@preact/signals-core').Signal>} */
const cells = new Map();

/**
 * Get or create a signal for a cell name.
 * @param {string} name
 * @returns {import('@preact/signals-core').Signal}
 */
export function getCell(name) {
  if (!cells.has(name)) {
    cells.set(name, signal(undefined));
  }
  return cells.get(name);
}

/**
 * Resolve a value that may be a cell reference.
 * Cell references are strings starting with "cell:".
 * @param {*} value
 * @returns {*} The resolved value (reactive if it's a cell reference)
 */
export function resolveValue(value) {
  if (typeof value === 'string' && value.startsWith('cell:')) {
    return getCell(value.slice(5)).value;
  }
  return value;
}

/**
 * Initialize cell listeners on the bridge.
 */
export function initCells() {
  onMessage('cell:register', (msg) => {
    const s = getCell(msg.name);
    s.value = msg.value;
    console.log(`[cells] Registered: ${msg.name} =`, msg.value);
  });

  onMessage('cell:update', (msg) => {
    const s = getCell(msg.name);
    s.value = msg.value;
  });
}
```

**Step 3: Create renderer.js — Lit-based primitive renderer**

Create `frontend/core/renderer.js`:
```javascript
/**
 * Dynamic renderer: converts a Racket primitive tree into DOM using Lit Web Components.
 * Each primitive type maps to a `mr-*` custom element.
 */
import { onMessage } from './bridge.js';

let rootContainer = null;

/**
 * Render a primitive tree node into a parent DOM element.
 * @param {object} node - Primitive tree node { type, props, children }
 * @param {HTMLElement} parent
 */
export function renderNode(node, parent) {
  if (!node || typeof node !== 'object') return;

  const tagName = `mr-${node.type}`;

  // Check if the custom element is registered
  if (!customElements.get(tagName)) {
    console.warn(`[renderer] Unknown primitive: ${tagName}`);
    const fallback = document.createElement('div');
    fallback.textContent = `[Unknown: ${node.type}]`;
    parent.appendChild(fallback);
    return;
  }

  const el = document.createElement(tagName);

  // Set properties from props
  if (node.props) {
    for (const [key, value] of Object.entries(node.props)) {
      el[key] = value;
    }
  }

  // Recursively render children
  if (node.children && Array.isArray(node.children)) {
    for (const child of node.children) {
      renderNode(child, el);
    }
  }

  parent.appendChild(el);
}

/**
 * Replace the entire layout.
 * @param {object} tree - Root primitive tree from Racket
 */
export function setLayout(tree) {
  if (!rootContainer) {
    console.error('[renderer] Root container not set');
    return;
  }
  rootContainer.innerHTML = '';
  renderNode(tree, rootContainer);
}

/**
 * Initialize the renderer.
 * @param {HTMLElement} container - The root DOM element to render into
 */
export function initRenderer(container) {
  rootContainer = container;

  onMessage('layout:set', (msg) => {
    console.log('[renderer] Setting layout');
    setLayout(msg.tree);
  });
}
```

```javascript
    content: { type: String },
    level: { type: Number },
  };

  static styles = css`
    :host { display: block; }
    h1 { font-size: 1.8rem; font-weight: 700; }
    h2 { font-size: 1.4rem; font-weight: 600; }
    h3 { font-size: 1.1rem; font-weight: 600; }

  constructor() {
    super();
    this.level = 1;
    this._dispose = null;
  }

  connectedCallback() {
    super.connectedCallback();
    // Set up reactive binding if content references a cell
    if (typeof this.content === 'string' && this.content.startsWith('cell:')) {
      this._dispose = effect(() => {
        this._resolvedContent = resolveValue(this.content);
        this.requestUpdate();
      });
    }
  }

  disconnectedCallback() {
    super.disconnectedCallback();
    if (this._dispose) this._dispose();

  get displayContent() {
    return this._resolvedContent ?? resolveValue(this.content);

    const text = this.displayContent;
    switch (this.level) {
      case 1: return html`<h1>${text}</h1>`;
      case 2: return html`<h2>${text}</h2>`;
      case 3: return html`<h3>${text}</h3>`;
```

**Step 1: Add menu handling to bridge.rs**

Add a new Tauri command and menu builder. Tauri v2 uses `tauri::menu::MenuBuilder`. The Racket side sends a menu structure; the Rust side builds native menus from it.

Add to `bridge.rs` or create `src-tauri/src/menu.rs`:

```rust
use tauri::menu::{MenuBuilder, SubmenuBuilder, MenuItemBuilder, PredefinedMenuItem};
use tauri::{AppHandle, Wry};
use serde_json::Value;

pub fn build_menu_from_json(app: &AppHandle, menu_data: &Value) -> Result<tauri::menu::Menu<Wry>, String> {
    let mut builder = MenuBuilder::new(app);

    if let Some(items) = menu_data.as_array() {
        for item in items {
            let label = item.get("label").and_then(|l| l.as_str()).unwrap_or("");
            let children = item.get("children").and_then(|c| c.as_array());

            if let Some(children) = children {
                let mut submenu = SubmenuBuilder::new(app, label);
                for child in children {
                    let child_label = child.get("label").and_then(|l| l.as_str()).unwrap_or("");
                    if child_label == "---" {
                        submenu = submenu.separator();
                    } else {
                        let item = MenuItemBuilder::with_id(
                            child.get("action").and_then(|a| a.as_str()).unwrap_or(child_label),
                            child_label
                        ).build(app).map_err(|e| e.to_string())?;
                        submenu = submenu.item(&item);
                    }
                }
                let submenu = submenu.build().map_err(|e| e.to_string())?;
                builder = builder.item(&submenu);
            }
        }
    }

    builder.build().map_err(|e| e.to_string())
}
```
